# Project

Project NodeJs with MongoDB


## Installation

```sh

npm init -y

npm i dotenv                    // DovEnv

npm i express                   // Express
npm i express-validator         // Validator fields

npm i mongoose                  // Base datos

npm i bcryptjs                  // Encriptar password

```


## Run Server

```sh

node app.js

nodemon app.js


```
